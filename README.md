# Devopress
Devopress WordPress Theme

## Get started:
```bash
cd wp-content/themes/devopress/
cp .env.example .env
make up
```

***

## Development:

```bash
cd wp-content/themes/devopress/
nvm use
npm run dev
```
***

## Build:

```bash
cd wp-content/themes/devopress/
nvm use
npm run build
```
